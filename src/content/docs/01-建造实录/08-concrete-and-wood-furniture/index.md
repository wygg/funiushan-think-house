# 🪚 自主家具系统：钢筋水泥与原木的手工生产线

> **核心哲学**：空间生产的最后一步，是彻底打破批量流水线家具的同质化，亲手设计并铸造不可替代的生活交互界面。

## 🎯 阶段实学目标
*   **结构力学与材料美学**：不仅是买家具，而是造物。Momo 将运用粗犷的钢筋水泥骨架，搭配原木台面，亲手铸造极具生命力与废墟美学的“厨房操作台”。
*   **工程统筹拉手**：从计算木材的方量、切割损耗，到钢筋笼的绑扎与混凝土现浇脱模时间，全流程演练一次微缩版的土木工程。

## 📂 结构说明
*   `01-research/`：AI 提供的脱模剂选择、水泥养护干湿度时间以及原木防腐涂装工艺研究。
*   `02-execution/`：手工铸造工艺流程 SOP。
*   `03-drawings/`：钢筋骨架结构设计图、木作倒角剖面图。
