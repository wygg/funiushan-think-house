# momo山居：低成本清水混凝土墙面（安藤忠雄风）施工执行方案

## 1. 方案背景与目标
* **背景**: 目前墙面为毛墙（已批水泥），触感拉手、略粗糙，但基本不掉灰。
* **目标**: 以极低成本（平替微水泥）实现“安藤忠雄”清水混凝土风 / 工业风质感。
* **核心需求**: 
  1. 视觉上有自然的斑驳感，保留纯正水泥质感。
  2. 触感平滑，不拉手。
  3. 绝对不掉灰，日常易于打理（可擦拭）。
* **预算约束**: 极低成本。

## 2. 方案选择矩阵
根据现阶段墙面基础的视觉美观度，提供两套梯次实施方案。

### 方案 A：极简固化（原汤化原食）
* **适用条件**: 评估现有毛墙的水泥斑驳感、颜色深浅已达到审美预期。
* **核心动作**: 物理打磨去粗糙 + 化学涂层封固。
* **预估成本**: 约 3-5 元/㎡。

### 方案 B：瓷砖胶基面重塑（高级平替法）
* **适用条件**: 现有毛墙基础颜色太难看、不均匀，或平整度不够。需要重新覆盖一层质感面。
* **核心动作**: 瓷砖胶混水泥薄批 + 半干压光 + 化学涂层封固。
* **预估成本**: 约 8-15 元/㎡。不推荐使用“普通腻子+水泥漆”组合，因其缺乏水泥真实的骨料质感，视觉显廉价。

---

## 3. 标准作业程序 (SOP)

### SOP - 方案 A（极简固化）
**BOM（物料清单）**: 
- 120目 或 240目 砂纸及砂纸架
- 纯水性哑光罩面漆（清水混凝土/微水泥专用）
- 吸尘器/鸡毛掸子、短毛滚筒刷

**施工步骤**:
1. **轻度打磨**: 将砂纸架装上杆子，对墙面进行全覆盖式“轻扫”打磨。目的仅为磨除表面锋利、拉手的水泥砂砾结晶，切勿重度打磨破坏原有斑驳肌理。
2. **彻底清灰**: 使用吸尘器或干鸡毛掸子清理所有墙面浮灰，确保表面无粉尘残留。
3. **两遍罩面**: 将水性哑光罩面漆倒入托盘，用滚筒像刷乳胶漆般滚涂第一遍。待第一遍表干后（约2-4小时），滚涂第二遍。
4. **终检**: 漆膜实干后，墙面会呈现“鸡蛋壳”般的微涩平滑手感，且具有防水防掉灰功能。

### SOP - 方案 B（瓷砖胶基面重塑）
**BOM（物料清单）**:
- 普通灰色瓷砖胶（粉料越细越好，例如协进超强力产品）
- 白水泥（用于调浅灰色度）或 灰水泥
- 纯水性哑光罩面漆
- 不锈钢泥工抹子、干净的铁抹子（用于收光）、批灰桶、电动搅拌器

**配比标准**:
- `瓷砖胶 : 水泥 = 2 : 1` 
- *配比原理：瓷砖胶提供高分子胶粉（高强粘结、防掉粉）及细石英砂骨料；水泥提供强度及基础成色。可通过增减白水泥比例调节灰度。*

**施工步骤**:
1. **基面微润**: 稍微润湿现有毛墙。
2. **调浆测试**: 按 2:1 比例加水用电动搅拌器搅拌成“芝麻糊”状。**强制要求**：先在一个不起眼的角落进行约 1㎡ 的打样，待其干透（湿润时颜色较深，干后会变浅）确认颜色符合“安藤灰”预期后，再大面积调配。
3. **极薄批刮**: 由泥工师傅使用不锈钢抹子，在墙面进行薄批（厚度控制在 1-2 毫米）。泥工师傅正常的批刮刀印及墙面吸水率差异，干透后会自动形成自然的流云斑驳感。
4. **半干压光（关键工艺）**: 待墙面达到“半干表面”（手指按不出明显深坑，但触感仍潮湿），必须换用干净的铁抹子，用力在墙面上来回“压光”（收光）。**此动作是解决拉手问题、形成紧实细腻高级质感的绝对核心**。
5. **养护与罩面**: 待重塑墙体彻底干透（约3-5天），完全按照方案A的步骤，滚涂 1-2 遍水性哑光罩面漆进行封层保护。

---

## 4. 实施建议与避坑指南
1. **严禁亮度**: 罩面漆极度关键，必须采购**纯哑光**型号（光泽度<5%）。亮光漆会使水泥墙反光，彻底破坏高级工业风。
2. **禁止添加沙子**: 在方案B中，严禁在调浆时私自掺入散沙，会极大增加整体粗糙度。
3. **手工属性**: 水泥天然具有不可控的色彩纹理，需包容并欣赏其随机产生的色差，这正是“Wabi-Sabi”（侘寂）与清水混凝土精神的精髓所在。
